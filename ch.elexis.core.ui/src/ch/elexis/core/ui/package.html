Steuer-Package

Dieses Package ist die zentrale Ablaufsteuerung des Plugin-Systems. Die Klasse Hub definiert verschiedene globale Variablen, die Klassen mit ..Perspektive definieren Perspektiven. Beim Programmstart werden die Steuerklassen in definierter Reihenfolge aufgerufen:

Aktivierungsreihenfolge

  • Hub
  • Desk
  • ApplicationWorkbenchAdvisor
  • ApplicationWorkbenchWindowAdvisor
  • ApplicationActionBarAdvisor
  • LoginPerspektive
  • PatientPerspektive
